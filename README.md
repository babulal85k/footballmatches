# Redux React Football Match List App

This is a simple React application that uses Redux for state management to display a list of football matches fetched from an API. It also integrates Chakra UI for styling.

## Features

- Displays a list of football matches fetched from an API
- Handles loading and error states while fetching data
- Allows pagination for a better user experience
- Built with Redux for state management and Chakra UI for styling

## Installation

1. Clone the repository:

```
git clone https://github.com/your-username/redux-react-football-match-list.git
```

2. Install dependencies:

```
cd redux-react-football-match-list
npm install
```

3. Run the development server:

```
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

## Usage

- Upon loading, the app fetches football matches from the API and displays them.
- You can see a loading message while data is being fetched.
- If there's an error fetching data, an error message is displayed.
- The matches are displayed in a list format.
- Pagination is available for navigating through the match list.

## Technologies Used

- React
- Redux
- Redux Thunk
- Chakra UI
- Axios
